## Get started

1. Install expo and eas globally.
   
   ```bash
   npm install --global expo-cli eas-cli
   ```
   
2. Install dependencies

   ```bash
   npm install
   ```

3. Configure firebaseConfig.js with new configuration

4. Start the app

   ```bash
    npx expo start
   ```
